헛걸음보상 서비스 이용방법

  1. 1환불 진행 시 차량의 상태가 출고시점과 상이할 경우온라인 예약 필수이며 판매딜러가 예약을 승인한 건에 한함
  2. 2예약시간에 맞춰 현장 방문
  3. 3허위매물사고 발생시, My Page > 관심차량관리에서 허위매물 신고
  4. 4딜러 및 고객을 통한 사고경위 확인
  5. 5신고일로부터 5일 이내 사고처리결과 및 보상금 지급여부 통지
  6. 6고객정보 확인 후 고객계좌로 보상금 지급 처리
