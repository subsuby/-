> 개발환경 셋팅
    1. 프로젝트 임포트
    2. 프로젝트 언어 설정
    3. 의존 라이브러리 다운로드 for Gradle
        "Navigator" View > 프로젝트 명에 마우스 우 클릭 > "Gradle" > "Refresh All" >> Console에 라이브러리 다운로드 진행 확인 및 "Success" 메시지 나오면 라이브러리 다운로드 완료
    4. 프로젝트 코드 주석 템플릿 설정 ("/project_doc/setting/codeTemplate/setupTemplate.txt" 확인)
    기타. 이클립스 UML 플러그인 설치
        상단 메뉴 > "Help" > "Install New Software" > URL 입력란에 "http://www.objectaid.net/update/" 검색 > 하단 체크박스 선택하여 설치 완료
    5. 로그 레벨 설정
        "src > main > resources > logback.xml" 에서 "root level"을 "debug"로, "appender-ref"를 "STDOUT"으로 설정하여 콘솔에 나오도록 한다.
